Politique de Confidentialité - SBC Easy

Dernière mise à jour : 2 Mars 2024

Introduction

Cette politique de confidentialité explique comment l'extension SBC Easy ("nous", "notre", "nos") collecte, utilise et protège vos informations lorsque vous utilisez notre extension Chrome.

Collecte et Utilisation des Données

SBC Easy est conçu pour fonctionner localement dans votre navigateur et n'effectue aucune collecte de données personnelles. L'extension :

  • N'accède qu'aux données nécessaires sur la page web de FIFA Ultimate Team pour résoudre les SBC
  • Ne collecte ni ne stocke aucune information personnelle
  • Ne transmet aucune donnée à des serveurs externes
  • Ne partage aucune information avec des tiers

Permissions du Navigateur

L'extension nécessite uniquement l'accès à la page web de FIFA Ultimate Team pour fonctionner. Cette permission est utilisée exclusivement pour :

  • Lire les informations des SBC disponibles
  • Analyser les joueurs de votre club
  • Proposer des solutions optimales pour les SBC

Stockage Local

Toutes les données temporaires nécessaires au fonctionnement de l'extension sont stockées localement dans votre navigateur et sont automatiquement effacées lorsque vous fermez l'onglet.

Modifications de la Politique de Confidentialité

Nous nous réservons le droit de mettre à jour cette politique de confidentialité à tout moment. Les modifications seront publiées sur cette page.

Contact

Si vous avez des questions concernant cette politique de confidentialité, vous pouvez nous contacter via le Chrome Web Store.
